# EcoTunga

## Project Structure
- `ecotunga_frontend/`: Frontend application
- `EcoTunga_backend/`: Backend application

## Setup Instructions

1. Install dependencies:
```bash
npm run install-all
```

2. Run the development servers:
```bash
npm run dev
```

git add .
